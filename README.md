# README

Code for _The stability of convection in the North Atlantic subpolar gyre in a conceptual model_ 

Notebooks starting with _fig__ were used to generate the figures in the article; the notebook table-R generates the data for the table summarizing the state ratio R for different noise values.

AuxiliaryFunctions/* contains the model and additional functions used for plotting.
Data/* contains the data that was generated for the heatmap.
Figures/* contains the figure files.
